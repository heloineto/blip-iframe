## Usage

The `hideNavbar` function is used to hide the top navigation bar of the Blip platform.
This can be useful if you want to create a full screen extension. To show the navbar again, use the `showNavbar` function.

## Example

```ts
import { blip } from 'blip-iframe';

await blip.hideNavbar();
```

## Parameters

The `hideNavbar` function accepts no parameters.

## Return type

The `hideNavbar` function does not return any value.
