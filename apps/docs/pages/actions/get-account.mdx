## Usage

The `getAccount` function is used to retrieve details about the current user account.
It provides information such as full name, email, phone number, photo URI, time zone name, culture, and creation date.

## Example

```ts
import { blip } from 'blip-iframe';

const response = await blip.getAccount();

if (!response.success) {
  console.error(response.error); // An error happened during the request
  return;
}

console.log(response.data); // The account details
```

## Parameters

The `getAccount` function accepts no parameters.

## Return type

The `getAccount` function returns the following response:

```ts
export interface GetAccountResponse {
  fullName: string;
  alternativeAccount: string;
  identity: string;
  email: string;
  phoneNumber: string;
  photoUri: string;
  timeZoneName: string;
  culture: string;
  creationDate: string;
}
```
